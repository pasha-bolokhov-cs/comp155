Assignment

  • For the following, you will need to use the Human Resource tables. These were created using the "HR_Start.sql" script from lab 1. You will be required to SPOOL the queries and results to a file so that you can print them to hand in. See pages 69-70 in your text or the Notes: section at the end of this lab.  These questions do not use the STOREDB tables used in the textbook examples, only the tables you installed from lab 1 (e.g. the Employees table). Don't forget to use the DESC Employees statement in SQL*Plus to view the structure of the Employees table.
  1. Write a query to display the last name and salary for all employees whose salary falls in the range of $2,500 to $2,600 inclusive. (Hint: use the BETWEEN operator.) Label the columns Poor Employee and Monthly Salary respectively. Note the use of upper and lower case in the column labels. Also note the space in each column label. Make sure these are shown in the labels.
  2. Write a query to display the last name and department number of all employees in departments 60, 70 and 90 in descending alphabetical order by last name. (Hint, use the IN operator)
  3. Write a query to display the last names of all employees where the third letter of their last name is an a.
  4. Write a query to display the first name, last name, salary, and commission for all employees who earn commissions and have a last name beginning with the letter S. Sort data in descending order of salary and commissions. (Hint, you must handle NULL values.)

Be sure to use proper formatting in your SQL statements. Your SELECT should be on one line, your FROM on another and so on. Each clause should start a new line. You will lose marks if you do not use proper formatting to your SQL statements!

Also, be sure to clearly indicate on the hard copy you hand in which query and result goes with which assignment number.

For example, this SELECT query is properly formatted:

SELECT last_name AS "Last Name", salary AS "Salary"
FROM employees
WHERE last_name = 'SMITH'
AND salary > 1000
ORDER BY salary;

This SELECT query is not properly formatted:

SELECT last_name,salary FROM customers WHERE last_name = 'SMITH' AND salary > 1000
ORDER BY salary;
